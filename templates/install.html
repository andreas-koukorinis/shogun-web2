Installing Shogun

For certain systems, we offer pre-built packages of Shogun. This is the easiest way to start using it. For other cases, see how to compile it from source code below.

Ready-to-install packages


You can directly install Shogun on the following systems.

Ubuntu ppa

First, add our prepackaged ppa as 
sudo add-apt-repository ppa:shogun-daily/ppa
sudo apt-get update
Then, install as 
sudo apt-get install libshogun17

Fedora

Shogun is part of Fedora 25

Docker images


You can run Shogun in our own cloud or set up your own using our Docker images as: 

sudo docker pull shogun/shogun
sudo docker run -it shogun/shogun bash

Building from source-code


There are various ways to build Shogun from source-code.

Python pypi

You can install from pipy. There is limited control over options and it might take a long time as everything is done from scratch. 
pip install shogun
